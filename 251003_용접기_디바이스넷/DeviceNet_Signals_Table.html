Lincoln Electric Power Wave DeviceNet Interface
Signal Specification

문서 정보: Y50031-21 DeviceNet Interface Specification (2010/04/20)
대상 장비: Power Wave 용접기 시리즈

1. Standard I/O Assembly Signals (표준 I/O 어셈블리)

1.1 Produced Assembly - Input to Power Wave (8 bytes)

Byte Bit Signal Name Type Description
0 0 Trigger Digital Welding trigger command (level sensitive)
0 1 Gas Purge Digital Gas purge command
0 2 Touch Sense Cmd Digital Touch sense command (level sensitive)
0 3 Cold Inch Fwd Digital Cold inch forward command
0 4 Cold Inch Rev Digital Cold inch reverse command
0 5 Weld Output Disable Digital Disable welding output
0 6 Fault Reset (Arc Obj) Digital Reset Arc Object faults (Arc Object only)
1 0 Schedule Select (Bit 0) Digital Schedule selection bit 0
1 1 Schedule Select (Bit 1) Digital Schedule selection bit 1
1 2 Schedule Select (Bit 2) Digital Schedule selection bit 2
1 3 Process Select Ready Digital Process selection ready signal
1 4 Attribute Request Digital DeviceNet attribute request bit
1 5 Prod Mon Fault Reset Digital Production monitoring fault reset
1 6 PM SN Request Digital Production monitoring serial number request
2-3 - Analog Reference 1 Analog (16-bit) Welding Workpoint (WFS/Current)
4-5 - Analog Reference 2 Analog (16-bit) Welding Trim (Voltage)
6-7 - Analog Reference 3 Analog (16-bit) Wave Control 1

1.2 Consumed Assembly - Output from Power Wave (8 bytes)

Byte Bit Signal Name Type Description
0 0 Arc Detect Digital Arc detection status
0 1 Touch Sensed Digital Touch sense detected
0 2 Gas Fault Digital Gas controller fault
0 3 Wire Fault Digital Wire drive fault
0 4 Water Fault Digital Water fault (Shutdown 1)
0 5 Inverter Fault Digital Power source fault
0 6 Wire Stick Digital Wire stick condition detected
0 7 DN Fault (Sys Fault) Digital DeviceNet/System fault
1 0 Process Select Ack Digital Process selection acknowledge
1 1 Limit Error Digital Analog reference out of limits
1 2 Weld Complete Digital Weld cycle completed
1 3 PM Fault Digital Production monitoring fault/alarm
1 4 Gas Purge On Digital Gas purge active status
1 5 Active Feed Head Digital Active wire feed head (0=Head1, 1=Head2)
1 6 Toggle Bit (Arc Obj Only) Digital Toggle bit for Arc Object (alternates each packet)
1 7 Attribute Response Digital DeviceNet attribute response bit
2-3 - Analog Feedback 1 Analog (16-bit) Actual Voltage
4-5 - Analog Feedback 2 Analog (16-bit) Actual Current
6-7 - Analog Feedback 3 Analog (16-bit) Actual WFS (Wire Feed Speed)

4. Arc Object for ACDC Systems (Class 100 Extended)

참고: Arc Object는 ACDC 용접 시스템에서만 사용되는 확장 클래스입니다.

4.1 Arc Object Additional Attributes

Attr ID Name Access Description
100 Arc Number RO Arc instance number (1-4)
101 Arc Polarity R/W Arc polarity setting (EN/EP/AC)
102 Arc Enable R/W Enable/disable this arc
103 Fault Code RO Arc-specific fault code
104 Fault Reset R/W Reset arc faults

4.2 Arc Object I/O Assembly (Extended Format)

Byte Bit Signal Name Description
0 6 Arc Fault Reset Reset arc-specific faults (Arc Object only)
1 6 Toggle Bit Toggle bit for Arc Object (alternates each packet)

5. Phase Generator Objects (Class 102/103)

참고: Phase Generator는 다중 아크 용접 시스템을 위한 고급 제어 기능입니다.
Attr ID Name Class Description
1 Phase Enable 102 Enable phase generator
2 Phase Offset 102 Phase offset angle (degrees)
3 Phase Frequency 102 Phase switching frequency (Hz)
4 Arc Assignment 1 102 Arc 1 phase assignment
5 Arc Assignment 2 102 Arc 2 phase assignment
6 Arc Assignment 3 102 Arc 3 phase assignment
7 Arc Assignment 4 102 Arc 4 phase assignment
1 Polarity A Enable 103 Enable polarity A phase
2 Polarity B Enable 103 Enable polarity B phase
3 Balance Control 103 AC balance control (-100 to +100)

6. System Interface Object (Class 104)

참고: System Interface는 용접 이송 속도 제어를 위한 인터페이스입니다.
Attr ID Name Access Description
1 Travel Speed R/W Welding travel speed (ipm)
2 Travel Speed Enable R/W Enable travel speed control

4 Jog Forward R/W Jog forward command
5 Jog Reverse R/W Jog reverse command

8. Sequencer State Definitions

참고: Sequencer는 용접 프로세스의 14개 상태를 제어합니다. 각 상태는 고유한 Workpoint, Trim, Wave 제어 값을 가집니다.
State Number State Name Description
0 Idle 대기 상태
1 Setup 설정 상태
2 Preflow 가스 프리플로우 (용접 전 가스 공급)
3 Strike 아크 착화 (초기 아크 발생)
4 Start 용접 시작 단계
5 Upslope 상승 구간 (전류 증가)
6 Weld1 주 용접 구간 1
7 Weld2 주 용접 구간 2
8 Downslope 하강 구간 (전류 감소)
9 Crater 크레이터 충진 (용접 종료부 처리)
10 Burnback 번백 (와이어 연소)
11 Postflow 포스트플로우 (용접 후 가스 공급)
12 Restrike 재착화 (아크 재점화)
13 Fault 고장 상태

9. Common Error Codes

Error Code Category Description (Korean)
0x01 Gas Fault 가스 제어기 고장
0x02 Wire Fault 와이어 송급 고장
0x04 Water Fault 냉각수 고장 (Shutdown 1)
0x08 Inverter Fault 인버터 전원부 고장
0x10 Wire Stick 와이어 스틱 감지
0x20 DeviceNet Fault DeviceNet 통신 고장
0x40 Limit Error 아날로그 입력 범위 초과
0x80 Production Monitor 생산 모니터링 알람/고장

10. DeviceNet Connection Summary

Connection Type Description Data Size Update Rate
Polled I/O 실시간 제어 연결 (Produced + Consumed Assembly) 8 bytes each 2-100 ms (configurable)
Explicit Messaging 파라미터 설정 및 조회 (Class 100, 101, 102, 103, 104, 105) Variable On-demand
