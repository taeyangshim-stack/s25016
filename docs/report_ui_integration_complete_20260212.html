6. 발견된 이슈 및 향후 주의사항
1. PlanA/PlanB Mechanical Unit명 차이
PlanA는 GantryRob_7, PlanB는 GantryRob입니다. 향후 PlanA 코드를 PlanB로 포팅할 때 MOC.cfg를 반드시 확인하여 Mechanical Unit 이름을 변경해야 합니다.
2. RAPID 파서 ASCII 제한
RAPID 파서는 ASCII 문자만 지원합니다. 코멘트에도 유니코드 특수문자(↔, →, ° 등)를 사용하면 구문 오류가 발생합니다.
3. 모듈 로드 타입 구분 (PROGMOD vs SYSMOD)
main() 함수가 포함된 모듈은 반드시 Program Module(PROGMOD)로 로드해야 합니다. System Module(SYSMOD)로 로드 시 Check Program은 통과하나 런타임에 main을 찾지 못합니다.
4. 태스크별 모듈 할당 주의
Rob2_MainModule(T_ROB2 전용)을 T_ROB1에 로드하면 targetdata RECORD 정의 충돌이 발생합니다. 백업 복원 후 각 태스크에 올바른 모듈만 로드되었는지 확인이 필요합니다.
