2. 현황 및 주요 기술적 문제점

  • [1호기] 용접기 교체에 따른 코드 재사용 불가: 기존 링컨(Lincoln) 용접기에서 EWM 용접기로 변경됨에 따라 아크웨어* 및 ArcLink XT** 기반의 기존 코드를 그대로 사용할 수 없는 상황 — 코드 재작성 또는 대체 모듈 적용 필요
  • [2호기] 용접기 미확보: 현재 2호기에 투입할 용접기가 준비되어 있지 않음
    • R450 용접기: 중소조선연구소(RIMS)로부터 대여 필요
    • S700 용접기: 대불산단에서 대여 필요
  • [2호기] 조달 계획 미확정: 상기 용접기 대여를 위한 구체적인 일정 및 방법이 아직 확정되지 않은 상태
  • 통신 불안정: 중소조선연구소(RIMS)***와의 소켓 통신 반응 속도 저하 및 프로그램 다운 현상 확인

* 아크웨어(ArcWare): ABB 로봇에서 용접 실행 시 사용하는 용접 제어 모듈
** ArcLink XT: ABB 로봇과 Lincoln 용접기 간 접속 통신 모듈
*** RIMS (중소조선연구소): 2호기 적용 회전 토치 개발 기관 — 소켓 통신으로 시스템과 연동

3. 호기별 상세 사양 및 결정 사항

【최종 결정】 본 회의에서 B라인 호기별 적용 플랜을 다음과 같이 확정하였음
  • 1번 거더 (1호기): 플랜 B 적용
  • 2번 거더 (2호기): 플랜 C 적용
구분 용접기 사양 토치 및 센싱 방식 적용 플랜 비고
1호기 EWM 용접기 프로니우스(Fronius) 상용 토치
직접터치식 용접모션 구현 예정		 플랜 B 개발 중 — 검증 필요
2호기 링컨(Lincoln) R450 + S700 (대여 예정) 회전 토치 + 더블 와이어 토치
비접촉 비전 센싱 (직접터치식 적용 불가)		 플랜 C 고도화 — 돌림용접 포함

세부 추진 전략

  • [1호기] 직접터치식 용접모션 구현: 프로니우스 상용 토치의 강성이 충분하므로 직접터치식 센싱 방식으로 용접 위치를 검출하여 자동 용접모션 구현
  • [2호기] 비접촉 비전 센싱 도입: 회전 토치는 구조적 강성이 부족하여 직접터치식 적용 시 내구성 문제로 시운전 자체가 불가능할 것으로 판단 — 비접촉 비전 카메라(1호기에서 이전 배치)를 통해 용접선 검출 방식 적용
  • [2호기] 공법 고도화: S700 기반 더블 와이어 동시 공급 및 Weld Guide 트래킹 적용으로 돌림용접 구현
  • 공정 최적화: 설비 특성에 맞춰 '한 판씩 순차 용접' 방식 채택

4. 모니터링 및 UI 구성

  • 데이터 가시화: 시작점/끝점(Start/End Point) 표시로 공정률 확인 간소화 — 실시간성 확보 (김병철 소장 담당)
  • 통합 관리: 별도 PC 없이 운영 소프트웨어 UI 내 3D 관제 뷰 통합 운영

5. 향후 일정 및 운영 계획

  • 현장 테스트: 4월 중순 전기 투입 및 I-Check 일정에 맞춰 기술 인력 투입
  • 데이터 정밀화: 2호기 적용을 위한 기존 RPM·매크로 데이터 업데이트 진행
