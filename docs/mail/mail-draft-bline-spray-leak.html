[S25016] B라인 스프레이 누유 현상 원인분석 및 조치계획 검토 요청

안녕하십니까.
에스피시스템스 심태양입니다.

B라인 불스아이 스프레이 누유 현상에 대한 원인분석 및 조치계획을 안내드리오니 검토 요청드립니다.

A라인 현황

스프레이 누유받이를 설치한 상태로 내용 관망 중

B라인 사전 조치 사항 (2호기) - 기 조치 완료건

이전 B라인 2호기에서 누유 현상이 발생하여 확인한 결과, 스프레이 분무 조절밸브가 과다 개방 상태였음
분무 조절밸브 조절하여 조치 완료된 상태
해당 분무 조절밸브는 7대 전 호기 조절 셋팅 완료 상태

B라인 누유 현상 분석 (1호기 집중 발생)

금일 현장 오퍼레이터 김복삼 기연님과 내용을 확인한 사항입니다.

▸ 전체 구성

▸ 스프레이 구성 (밸브 / 분무홀 / 분무가이드)

▸ 원인분석

직전 시퀀스인 용접토치 클리닝 시 발생한 가루(이물질)가
분무(공압) 시 분사액과 섞여 분무가이드 내 단차에 쌓임
→ 분무(안개)를 액상으로 머금고 있다가 액상이 모여 하부로 떨어지는 상황

긴급 조치 (현재 상태)

전 호기 I/O 조치하여 분사 시퀀스 전체 제거 완료

조치 계획

분무가이드 내 단차를 제거하여 이물질 축적 방지

- 한 호기만 우선 적용하여 문제점에 대한 조치 효과 확인 예정
- 결과 확인 후 전 호기 확대 적용 계획

상기 내용 검토 부탁드립니다.

※ 현장 오퍼레이터 김복삼 기연님과 내용 공유 완료
