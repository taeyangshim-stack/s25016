1단계: Google Sheets 생성

1.1 새 스프레드시트 만들기

  1. Google Sheets (sheets.google.com)에 접속
  2. "새로 만들기" 또는 "+ 빈 스프레드시트" 클릭
  3. 스프레드시트 이름을 "S25016 근무관리"로 변경
✅ 완료! 이제 스크립트를 설정할 준비가 되었습니다.

2단계: Google Apps Script 설정

2.1 Apps Script 편집기 열기

  1. Google Sheets 메뉴에서 확장 프로그램 → Apps Script 클릭
  2. 새 탭에서 Apps Script 편집기가 열립니다
  3. 기본 function myFunction() {} 코드가 보입니다

2.2 스크립트 코드 복사

근무관리 폴더의 GoogleAppsScript.js 파일을 열어서 전체 내용을 복사합니다.

📂 파일 위치: s25016/근무관리/GoogleAppsScript.js

2.3 스크립트 붙여넣기

  1. Apps Script 편집기의 기존 코드를 모두 삭제
  2. 복사한 코드를 붙여넣기
  3. 왼쪽 상단의 저장 아이콘 클릭 (💾)

2.4 설정 수정

코드 상단의 CONFIG 영역을 수정합니다:

const CONFIG = {
  // 이메일 수신자 (쉼표로 구분)
  EMAIL_RECIPIENTS: 'your-email@example.com',  // ← 여기를 수정!

  // 참조(CC) 수신자 (선택사항)
  EMAIL_CC: '',  // 필요시 'cc@example.com'

  // 발신자 이름
  EMAIL_FROM_NAME: 'S25016 근무관리 시스템',

  // 시트 이름
  SHEET_NAME: '출입기록',

  // 메일 발송 여부 (true/false)
  SEND_EMAIL: true  // false로 설정하면 메일 발송 안 함
};
⚠️ 주의: EMAIL_RECIPIENTS를 본인의 이메일 주소로 꼭 변경하세요!

2.5 저장 및 테스트

  1. 상단의 저장 버튼 클릭
  2. 함수 선택 드롭다운에서 testSaveRecord 선택
  3. 실행 버튼 클릭
  4. 처음 실행 시 권한 승인 필요:
    • "권한 검토" 클릭
    • Google 계정 선택
    • "고급" → "안전하지 않은 페이지로 이동" 클릭
    • "허용" 클릭
✅ 테스트 성공! Google Sheets에 "출입기록" 시트가 생성되고 테스트 데이터가 추가되었습니다.

3단계: 웹 앱으로 배포

3.1 배포 시작

  1. Apps Script 편집기 우측 상단의 배포 → 새 배포 클릭
  2. "유형 선택" 옆 톱니바퀴 아이콘 클릭
  3. 웹 앱 선택

3.2 배포 설정

  • 설명: "S25016 근무관리 API v1" (선택사항)
  • 다음 사용자로 실행: 나
  • 액세스 권한: 모든 사용자
⚠️ 중요: "액세스 권한"을 반드시 "모든 사용자"로 설정해야 합니다!

3.3 배포 완료

  1. 배포 버튼 클릭
  2. 권한 승인 (이전 단계와 동일)
  3. 웹 앱 URL이 표시됩니다
웹 앱 URL:
https://script.google.com/macros/s/YOUR_SCRIPT_ID_HERE/exec
✅ 중요: 이 URL을 복사해두세요! 다음 단계에서 사용합니다.

🔧 문제 해결

❌ 데이터가 저장되지 않아요

  1. Script URL 확인: 입력.html의 SCRIPT_URL이 올바른지 확인
  2. 배포 설정 확인: "액세스 권한"이 "모든 사용자"로 설정되었는지 확인
  3. 브라우저 콘솔 확인: F12 → Console 탭에서 에러 메시지 확인

❌ 이메일이 오지 않아요

  1. CONFIG 확인: EMAIL_RECIPIENTS가 올바른 이메일인지 확인
  2. SEND_EMAIL 확인: true로 설정되어 있는지 확인
  3. 스팸함 확인: 스팸 폴더에 들어갔을 수 있습니다
  4. Apps Script 로그: Apps Script 편집기에서 실행 로그 확인

❌ 권한 오류가 발생해요

  1. Apps Script 편집기에서 다시 권한 승인
  2. "고급" → "안전하지 않은 페이지로 이동" 클릭
  3. 모든 권한 허용

💡 Apps Script 로그 확인 방법
