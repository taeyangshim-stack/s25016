■ 주요 개선 내용
================================================================================

1. 레이아웃 재구성
   - 3열 구성으로 기능별 영역 명확히 분리
   - 좌측: 좌표계 이미지 + 배속 선택
   - 중앙: 위치 이동 테이블 (X/Y/Z 버튼 + 위치값)
   - 우측: 이동 간격 선택

2. 축별 색상 구분
   - X축 (빨강): 좌우 이동
   - Y축 (초록): 전후 이동
   - Z축 (파랑): 상하 이동

3. 좌표계 이미지 추가
   - 실제 장비 사진에 축 방향 표시
   - 직관적인 방향 인식 가능

4. 리밋 경고 표시
   - 10% 이내 접근 시: 노란색 경고
   - 5% 이내 접근 시: 빨간색 위험
