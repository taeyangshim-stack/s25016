## 🔧 문제 해결

### "이슈 등록에 실패했습니다"

**원인**: SCRIPT_URL이 올바르지 않음

**해결**:
1. `punchlist/scripts/punchlist.js` 6번째 줄 확인
2. Apps Script 배포 URL과 정확히 일치하는지 확인
3. URL이 `/exec`로 끝나는지 확인
4. 따옴표 안에 공백이 없는지 확인

### "이슈를 불러오는 데 실패했습니다"

**원인**: Google Apps Script 권한 문제

**해결**:
1. Apps Script > **배포 > 배포 관리**
2. 현재 배포 옆 ⋮ 메뉴 > **수정**
3. **액세스 권한: 모든 사용자** 확인
4. **새 버전** 생성 후 재배포

### Google Sheets에 데이터가 저장되지 않음

**원인**: SHEET_ID 오류

**해결**:
1. Google Sheets URL에서 SHEET_ID 재확인
2. Apps Script에서 `SHEET_ID` 상수 업데이트
3. 저장 후 재배포
