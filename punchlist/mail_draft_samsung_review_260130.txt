■ 1. 갠트리 조깅 UI 개선안 (PL-2026-170)

▶ 데모 페이지:
https://s2501602.vercel.app/punchlist/3week-plan/gantry-jog-ui-demo.html

※ 유의사항:
- 데모 버전은 HTML로 구성되어 있어 실제 적용 시 색상이 다를 수 있습니다
- 버튼 구성 및 기능 배치에 대해 확인 부탁드립니다

▶ 주요 개선 내용:

1) 속도 조절 기능 추가
   - 25%, 50%, 75%, 100% 배속 버튼 추가
   - 좌측 하단 배치 (좌표 표시 영역 아래)

2) 위치 정보 표시 개선
   - 최대위치, 현위치, 최소위치 순서로 배치
   - X, Y, Z 축별 실시간 좌표 표시

3) 축 방향 표시 추가
   - X축: 좌우 방향 (← →) / 왼쪽이 +
   - Z축: 상하 방향 (↑ ↓) / 위쪽이 +
   - Y축: 좌측하단 → 우측상단 (↗) / 좌측하단이 +

4) 불필요 요소 제거
   - 이동간격 하단 "100" 표시 (흑색 배경) → 제거
   - "최소위치" 위 빈 공간 → 제거

▶ 검토 요청 사항:
- 버튼 레이아웃 적절성
- 위치 정보 표시 순서 (최대→현재→최소)
- 속도 조절 버튼 위치


■ 2. 공압/가스 알람 처리 기준 (PL-2025-067)

1/29(목) 협의한 내용을 정리하였습니다.
아래 기준으로 적용해도 되는지 검토 부탁드립니다.

▶ 1) 공압 (Air Pressure)
┌──────────┬────────────────────┬──────────┐
│ 구분     │ 기준               │ 처리     │
├──────────┼────────────────────┼──────────┤
│ 정상치   │ 6 bar              │ -        │
│ 상한치   │ 제한 없음          │ -        │
│ 알람     │ 5 bar 이하 5초 유지│ ⚠️ 알람  │
│ 에러     │ 3 bar 이하 5초 유지│ 🛑 에러  │
└──────────┴────────────────────┴──────────┘

▶ 2) 가스 (CO2 Gas)
┌──────────┬─────────────────────────┬──────────┐
│ 구분     │ 기준                    │ 처리     │
├──────────┼─────────────────────────┼──────────┤
│ 에러     │ 0.5 bar 이하 5초 유지   │ 🛑 에러  │
└──────────┴─────────────────────────┴──────────┘

▶ 커스텀 알람 팝업 개선 사항:
- 기존: 폰트 32 고정 → 창 축소 시 알람 내용 잘림 발생
- 개선: 창 크기에 따라 폰트 자동 조절 (가독성 확보)
- 중복 멘트 "다음 알람이 발생했습니다:" 삭제
- 해당 알람만 표시 (단독/동시 발생 모두 지원)
- 핵심 키워드 볼드 강조 처리로 가독성 향상

▶ 알람 팝업 데모 페이지:
https://s2501602.vercel.app/punchlist/3week-plan/alarm-popup-demo.html
