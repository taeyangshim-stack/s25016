문제 상황
B라인 갠트리 X축 브레이크 제어 시, X1(주축)과 X2(종동축)의 Control Off Speed Limit 설정값이 상이하여 브레이크 동작 타이밍 불일치 발생 가능성 확인
원인 분석
축 기존 설정 수정 설정 설명
X1 (주축) 0.02 (2%) → 0.02 (2%) 유지
X2 (종동축) 1 (100%) → 0.02 (2%) 수정
문제: X2축이 100% 속도에서 바로 브레이크 → X1축과 동기화 불일치
조치 내용
X1, X2 축 모두 0.02 (2%) 속도까지 감속 후 브레이크 작동하도록 통일
! B라인 Control Off Speed Limit 설정 ! 파일: 1200-526406 (B3 로봇) X1_ControlOffSpeedLimit := 0.02; ! 주축: 2%까지 감속 후 브레이크 X2_ControlOffSpeedLimit := 0.02; ! 종동축: 2%까지 감속 후 브레이크 (기존 1 → 0.02 수정)
조치 결과
  • X1/X2 축 브레이크 동작 타이밍 동기화
  • 2% 속도 감속 → 브레이크 작동 순서 통일
  • 갠트리 정지 시 축간 위치 편차 최소화
