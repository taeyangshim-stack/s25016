X1/X2 브레이크 딜레이 튜닝 계획

비상정지 시 브레이크 즉시 체결로 인한 요동(oscillation)과 충격을 줄이기 위해 Brake Control On Delay를 단계적으로 조정하여 최적값을 탐색합니다.
기본 정보 주의사항 테스트 절차 결과 기록 체크리스트

1. 기본 정보

고정값Control Off Speed Limit = 0.02

  • 방향: 점진적 감소

2. 주의사항

⚠️ E-Stop 안전 규정 검토 미완료
본 테스트는 "기능 확인" 목적이며, 최종 적용 전 반드시 안전 규정 검토가 필요합니다. 안전 기능의 의도적 지연/완화는 안전 규정/리스크 평가 범위에서만 수행해야 합니다.
테스트 전 필수 확인
  • 컨트롤러 백업 완료 여부
  • 현재 디폴트 파라미터 값 기록
  • 주변 작업자 대피 및 안전 구역 확보

3. 테스트 절차

최적값 선정 기준
  1. 밀림양 ≤ 250mm 이내
  2. 요동/진동이 허용 수준 이하
  3. 알람/경고 증가 없음

5. 작업 체크리스트

사전 준비

  • 컨트롤러 백업 완료
  • 현재 디폴트 파라미터 값 기록
  • 영상 촬영 장비 준비
  • 밀림양 측정 도구 준비 (줄자/마킹)
  • 안전 구역 확보 및 주변 작업자 대피

테스트 진행

  • Step-0: 디폴트 값 기준점 측정 (3회)
  • Step-1: 15.0s 테스트 (3회)
  • Step-2: 8.0s 테스트 (3회)
  • Step-3: 4.0s 테스트 (3회)
  • Step-4: 2.0s 테스트 (3회)
  • Step-5: 1.0s 테스트 (3회)
  • Step-6: 0.5s 테스트 (3회)

결과 정리

  • 최적값 선정 완료
  • 테스트 결과 문서화
  • 영상 파일 저장
  • 최종값 컨트롤러 적용

6. 롤백 계획

  1. Step 진행 중 NG 발생 시: 직전 OK Step으로 즉시 복귀
  2. 지속 NG 시: 초기 백업값으로 원복 후 원인 분석
  3. 원복 후 동일 시나리오 재시험으로 "원상태" 재현 확인
