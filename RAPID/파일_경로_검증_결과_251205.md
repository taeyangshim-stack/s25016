### RobotStudio HOME:/ 경로란?

RobotStudio의 **HOME:/ 디렉토리**는 가상 컨트롤러의 홈 디렉토리입니다.

**실제 위치 (Windows)**:
```
C:\Users\<사용자명>\Documents\RobotStudio\Systems\<시스템명>\HOME\
```

**실제 위치 (리눅스/WSL - 해당 없음)**:
RobotStudio는 Windows 전용 소프트웨어이므로 WSL 환경에서는 HOME:/ 디렉토리가 생성되지 않습니다.

**중요**:
- ❌ 이 파일들은 현재 Git 저장소의 `/home/qwe/works/s25016/` 경로에 생성되지 **않습니다**
- ✅ RobotStudio 시뮬레이션 실행 시에만 RobotStudio의 가상 컨트롤러 내부에 생성됩니다
- ✅ RobotStudio에서 "File Transfer" 기능으로 로컬 PC로 복사 가능

### 1. Git 저장소와 RobotStudio 분리

### 2. 테스트 결과 파일 수집 방법
RobotStudio에서 테스트 실행 후:

1. **RobotStudio File Transfer 사용**:
   - Controller > File Transfer
   - HOME:/ 디렉토리 접속
   - `*.txt` 파일들을 로컬 PC로 복사

2. **Git 저장소에 보관** (선택):
   ```bash
   # 테스트 결과 폴더 생성
   mkdir -p RAPID/test_results/251205_home_position_test

   # RobotStudio에서 복사한 파일 저장
   cp <다운로드_경로>/*.txt RAPID/test_results/251205_home_position_test/

   # Git 커밋
   git add RAPID/test_results/
   git commit -m "test: Add home position test results (2025-12-05)"
   ```

### 3. 문서 업데이트 필요 시
