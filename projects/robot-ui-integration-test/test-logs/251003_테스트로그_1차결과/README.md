# 251003 테스트로그 1차결과

- 목적: ROBOT↔UI 1차 결과 관련 로그/증빙 일괄 보관
- 업로드 지침
  - 파일명 규칙: `YYMMDD_작성자_항목명_설명.확장자`
  - 민감정보(개인/설비식별자)는 마스킹 처리
  - 대용량 영상은 사내 공유드라이브 경로 기재 + 텍스트 프록시 업로드

## 디렉터리
- `실행_부팅/` 부팅/자동실행/창상태 관련 캡처 및 로그
- `이동_리미트/` 리미트 초과/모드 가시성/용어 관련 화면·이벤트
- `안전_근접/` X축 안전거리 파라미터 및 검증 관련 자료
- `충돌검사/` 경로 사전검증 재현 로그, 스택, 케이스 정의
- `UI_알람/` 근접/리미트 UI 알람 미표시 재현 자료
- `속도_배율/` 속도·배율 설정 값, 적용 전/후 스냅샷
- `에러처리/` 재시도 정책 관련 로그, 실패시 복귀 동작 기록
- `와이어/` 교체 모드·커팅 동작 증빙
- `UI_가시성/` 오버레이/방향 표시 캡처
- `시뮬_실기/` 시뮬레이션 vs 실기 트레이스 비교 자료

## 권장 첨부물
- UI 콘솔 로그(.txt), 컨트롤러 이벤트(.csv), 파라미터 스냅샷(.json), 스크린샷(.png), 트레이스(.csv)
