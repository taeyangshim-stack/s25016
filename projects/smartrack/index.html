04
컨셉 A · MVP

핀-그리드 + 수동 가변

추천 지지점 자동 계산 → 작업자가 가이드에 따라 서포트 핀을 배치. 구조 단순, 성과 빠름.

핵심 구조

  • 랙 면에 촘촘한 홀 그리드
  • 서포트 핀 퀵 체인지
  • 시스템이 "권장 지지점" LED/AR 표시

장단점

  • 장점: 저비용, 고신뢰, TRL 빠른 상승
  • 단점: 완전 무인 어려움
추천: 1차년도 MVP로 현장 신뢰 확보
05
컨셉 B · 차별화

전동 서포트 + 반자동

서포트가 높이/각도를 자동 조절. 적치는 크레인/AMR + 센서 보조 도킹.

핵심 구조

  • 전동 스크류잭 / 리니어 액추에이터
  • 상단 패드: 회전(2축) + 미끄럼방지
  • 로드셀 하중 감지, 접촉 감지

장단점

  • 장점: 자동화 체감 큼, 곡면 대응
  • 단점: 모듈 단가, 유지보수
전략: 2차년도 현장 실증, A에서 입증된 알고리즘 그대로 적용
06
컨셉 C · 세계 최초급

셀 기반 Self-Reconfiguring

랙 자체가 셀 배열로 구성. 부재 인식 후 셀이 자동으로 안정 자세를 형성합니다.

핵심 구조

  • 다점 셀(10×N) 각각 높이 제어(서보)
  • 부재 3D/CoG 기반 셀 높이/접촉점 최적화
  • 하중 분포 실시간 모니터링 → 위험 시 재분배

장단점

  • 장점: "형상 무관" 정점, 특허/논문
  • 단점: 비용, 인증 난이도
전략: 3차년도 PoC/특허 트랙으로 병렬 추진
