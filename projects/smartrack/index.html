01 과제 개요 / 범위 RFP 해석 포인트

과제 프레이밍

중대형 선박 유연생산시스템 도입을 위한 지능형 물류 관리 기술개발 관점에서, 스마트랙은 "단일 장비"가 아니라 소조립 공장 전체 물류 관리 시스템의 핵심 노드로 정의합니다.

RFP 모호점 정리
"원자재 입출고"와 "조립장 내 물류 이송"이 별도 시스템으로 기술되어 스마트랙의 범위가 임시 보관 버퍼인지, 통합 물류인지 애매 → "조립장 내" + 입고~출고 전 과정 포괄 문맥을 근거로 통합 물류로 해석.

4대 세부 시스템

세부 시스템핵심 요소
지능형 원자재 입출고 관리지능형 랙 + 입출고 관리 S/W + 통합 연계
조립장 내 물류 이송 자동화AMR H/W + 자율주행 + 입출고 시스템 연동
완성품 품질관리 및 적재변형 계측 + 고중량 소조블록 + 지능형 적재 랙
디지털트윈 통합 관리생산공정 관리 + 디지털트윈 + 모니터링

※ 본 보고서는 "지능형 랙(유연 적치)"을 중심으로 AMR/크레인 및 디지털트윈 연계를 고려해 확장 아이디어를 담습니다.

소조립 공장 물류 흐름도 — SP시스템스 스마트랙 범위

녹색 영역이 스마트랙 핵심. 입고 스캔 ~ 완성품 출하까지 SP시스템스 담당.
주황 경로: 소조립 완료 후 완성 소조블록을 천장 크레인(중량물) 또는 AMR(경량 부재)로 스마트랙 출고 영역에 반송. 블록 무게(2~15t)에 따라 이송 수단이 결정되며, 도킹/위치지정은 스마트랙 시스템이 관리합니다.

02 스마트랙 정의 + 대상 부재 분류 부재 유형별 스펙

정의 (운영 관점)

스마트랙은 소조립 공장 내 절단 소재의 입고·보관·공급 및 완성 소조블록의 적재·출하까지를 관리하는 공장 전체 물류 관리 시스템이며, AMR과 연계해 공장 내 자재 이동을 자동화하고 CAD/생산스케줄과 연동하여 적시적소(JIT) 자재 공급을 실현합니다.

입고 → 적치 → 피킹 → 공정공급 완성 → 적재 → 출하 형상/무게/CoG 다양

대상 부재 분류 체계 (4개 패밀리)

상위 20% 유형이 물동량의 약 80%를 차지합니다.

유연 적치의 핵심 도전: 판재+보강재가 물동량 70%를 차지하지만, 소조블록은 형상/CoG가 가장 복잡하고 무게가 크므로 안전검증 난이도가 높습니다. → 1차년도 판재·보강재 중심 MVP, 2차년도부터 소조블록 확장 권장.
왜 "유연 적치(Free-Form)"여야 하나?

현실 제약

  • 형상·길이·폭·두께·CoG가 제각각
  • 공정 변동으로 재배치(reshuffling) 빈번
  • 안전(전도/좌굴) 요구가 높아 경험 의존

유연 적치의 이득

  • 공간 효율: "지지점" 기반 고밀도 적치
  • 운영 유연성: 적치 전략 자동 변경
  • 안전 내재화: 센서+모델 기반 표준화
03 유연 적치 스마트랙 아이디어 "부재가 랙을 정의"
핵심 문장 — 고정 구획(슬롯)이 아니라, 3D 형상 + CoG + 허용 변형/지지 조건에 따라 랙이 지지점(서포트)을 재구성하고, 시스템이 안전·접근성·공정연계성을 동시에 최적화합니다.

컨셉 A 핀-그리드 + 수동 가변 (MVP)

추천 지지점 자동 계산 + 가이드(작업자/크레인)로 시작. 구조 단순, 성과 빠름.

  • 랙 면에 촘촘한 홀 그리드 (100~150mm 피치)
  • 서포트 핀을 퀵 체인지로 위치 변경
  • 시스템이 "권장 지지점" 표시(LED/AR/라벨)
  • 장점: 저비용·고신뢰 / 단점: 완전 무인 어려움
TRL 빠른 상승인력 개입 일부

컨셉 B 전동 서포트 + 반자동 (차별화)

서포트가 높이/각도를 자동 맞춤. 적치는 크레인/AMR + 센서 보조 도킹.

  • 서포트: 전동 스크류잭/리니어액추에이터
  • 상단 패드: 회전(2축) + 미끄럼 방지 + 교체형
  • 센서: 로드셀, 접촉 감지, (선택) 변형 게이지
  • 장점: 자동화 체감 큼 / 단점: 모듈 단가 관건
자동 정렬/수평화모듈 신뢰성

컨셉 C 셀 기반 Self-Reconfiguring (세계 최초급)

랙 자체가 셀로 구성, 부재 인식 후 셀이 자동으로 안정 자세를 형성.

  • 다점 셀(10×N) 각각 높이 제어(서보/유압)
  • 부재 3D/CoG 기반 셀 높이/접촉점 최적화
  • 하중 분포 실시간 모니터링 → 위험 시 재분배
  • 장점: "형상 무관" 정점 / 단점: 비용·인증 난이도↑
특허/논문 강점개발 난이도 높음

"조선형" 차별화 포인트

  • 생산계획 연동: 3일 내 투입품은 접근성 최우선, 2주 대기품은 고밀도
  • 변형/품질: 허용 변형을 지키는 지지 설계
  • AI Yard Brain: 크레인 동선·재배치 비용까지 최적화
추천: 1차년도 A→B로 성과/현장수용성 확보, 병렬로 C의 핵심(셀 제어/특허)을 PoC로 분리.

05 인식 / 안전 알고리즘 S/W가 차별화

데이터 파이프라인

  1. 입고 시 3D 스캔(또는 CAD 매칭) → 메쉬/포인트클라우드
  2. 부재 무게/CoG 추정(실측 + CAD 결합)
  3. 적치 후보 자세 생성(회전/뒤집기 제한)
  4. 지지점 후보(접촉 가능 면/리브 우선)
  5. 안전/변형/접근성 점수로 최적 조합 선택
  6. 실적치 후 로드셀/변형 센서로 검증·학습
초기에는 규칙+검증센서로 현장 신뢰를 만든 뒤, 데이터가 쌓이면 최적화 강화.

06 AMR·크레인 인터페이스 도킹 / 통신 규격

도킹(정렬) 방식

  • 패시브 마커: QR/AprilTag + 카메라 (저비용)
  • LiDAR 기반: 반사판/형상 정렬 (분진 대응)
  • 기계 가이드: 콘/레일 최종 위치 결정 (가장 강건)
조선 환경(분진/오염/조명 변화)에서는 "기계 가이드 + 센서 보조" 조합이 안정적입니다.

통신/제어 골격

  • 상위: WMS/WCS ↔ 스마트랙 제어 (REST/MQTT)
  • 하위: PLC ↔ 서포트 모듈 (필드버스)
  • 안전: E-Stop, 인터록, 과하중/편하중 트립

도킹 방식·통신 프로토콜·이재 방법을 조기 합의하여 병렬 개발 가능하게 합니다.

결정해야 할 5가지

  1. 대상 부재 범위: "절단소재" vs "소조블록" 우선순위
  2. 자동화 강도: A(추천+수동) → B(전동) → C(셀)
  3. 도킹 방식: 기계 가이드 vs 비전/LiDAR
  4. 안전 기준: 허용 변형/지지 조건 확정
  5. 성능지표: 공간효율, 시간, 재배치 비용, 안전

추천 결론

현장성과 + 세계 최초급 스토리를 동시에 잡으려면 A로 빠르게 실증 → B로 차별화 → C는 PoC/특허 트랙으로 병렬 추진.

결국 차별화는 형상/CoG 기반 지지점 결정 + 안전 검증 + 생산계획 연계에서 나옵니다.
