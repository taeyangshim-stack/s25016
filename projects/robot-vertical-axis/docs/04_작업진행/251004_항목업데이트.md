- **내용**: PC ↔ 로봇컨트롤러 통신 에러 관련 알람 처리 개선
- **상태**: 개선필요

### 2. 갠트리 접근 UI 알람 처리
- **작업주체**: UI
- **A라인/B라인**: 공통
- **내용**:

  - 로봇 알람 중 UI에 표시할 알람 리스트 정리 필요

### 3. r축 충돌 방지(y 연동 리미트)

### 4. 이동 범위 초과 시 에러 처리

- **A라인**: 확인 필요

### 5. 초기 화면 최대화
