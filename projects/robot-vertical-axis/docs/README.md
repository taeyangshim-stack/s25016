# 250917_상하축이슈 프로젝트

## 프로젝트 개요
상하축 관련 이슈 해결을 위한 협업 프로젝트 (3명 동시 작업)

## 폴더 구조

### 📋 01_문제점분석
- 현재 상황 분석
- 문제점 정의 및 우선순위
- 원인 분석 자료

### 🛠️ 02_대책수립
- 해결 방안 검토
- 기술적 대안 평가
- 실행 계획 수립

### 📅 03_일정관리
- 프로젝트 일정표
- 마일스톤 관리
- 진행 상황 추적

### 🔄 04_작업진행
- 개별 작업 현황
- 진행률 보고
- 이슈 트래킹

### 📚 05_문서자료
- 기술 문서
- 참고 자료
- 회의록

### 🤝 99_공유폴더
- 팀 공통 파일
- 임시 작업 공간
- 의견 교환

## 협업 규칙

1. **파일명 규칙**: `YYMMDD_작성자_제목.확장자`
2. **수정 시**: 원본 백업 후 수정
3. **공유 전**: 99_공유폴더에서 검토 후 해당 폴더로 이동
4. **일일 업데이트**: 04_작업진행 폴더에 진행 상황 기록
